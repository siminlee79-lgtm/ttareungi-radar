개인정보처리방침

최종 업데이트: 2026년 5월 25일

이 문서는 따릉이 레이더 PWA 프로토타입 운영을 위한 초안입니다. 앱스토어 출시, 회원가입, 서버 푸시, 광고 또는 분석 도구를 적용하기 전 실제 운영 방식에 맞춰 다시 검토해야 합니다.

1. 서비스 성격

따릉이 레이더는 서울 열린데이터광장에서 제공하는 공공 API를 활용해 대여소 정보, 대여 가능 수량, 대체 대여소 정보를 보여주는 비공식 정보 제공 서비스입니다. 서울시, 서울시설공단, 서울자전거 따릉이의 공식 서비스가 아닙니다.

2. 수집하는 정보

  • 현재 위치 정보: 주변 대여소를 계산하기 위해 브라우저 권한 허용 시에만 사용합니다.
  • 저장 장소 정보: 사용자가 입력한 장소명, 주소, 좌표, 알림 시간을 브라우저 저장소에 저장합니다.
  • 알림 권한 정보: 브라우저 알림 허용 여부를 확인합니다.
  • 서비스 이용 로그: 향후 분석 도구를 붙이는 경우 방문, 클릭, 오류 로그 등이 수집될 수 있습니다.

3. 이용 목적

  • 현재 위치와 저장 장소 주변의 대여소 상태 확인
  • 저장 장소 기준 레이더 카드와 대체 대여소 추천
  • 사용자가 설정한 알림 시간 안내
  • 서비스 오류 확인과 기능 개선

4. 보관 기간

현재 PWA 단계에서는 저장 장소와 알림 시간이 사용자의 브라우저 저장소에 보관됩니다. 사용자가 브라우저 방문 기록, 캐시, 사이트 데이터를 삭제하면 해당 정보도 함께 삭제될 수 있습니다. 서버 DB를 도입하는 경우 별도 보관 기간과 삭제 방법을 고지합니다.

5. 제3자 제공 및 위탁

현재 초안 기준으로 사용자의 저장 장소 정보를 제3자에게 판매하거나 제공하지 않습니다. 다만 지도 표시, 공공 API 조회, 배포 인프라 운영을 위해 카카오 지도, 서울 열린데이터광장, Cloudflare Pages 등의 외부 서비스가 사용될 수 있습니다.

6. 위치정보 안내

위치정보는 주변 대여소를 계산하기 위한 목적으로만 사용합니다. 현재 단계에서는 서버에 위치정보를 저장하지 않으며, 브라우저 권한을 거부하면 현재 위치 기반 기능이 제한될 수 있습니다. 앱 출시 전 위치정보 이용약관을 별도로 마련할 예정입니다.

7. 문의

개인정보, 위치정보, 서비스 오류 관련 문의는 운영자 이메일로 접수합니다. 실제 도메인과 이메일을 확정한 뒤 이 항목을 갱신합니다.

문의 이메일: bizmanlee56@gmail.com

8. 데이터 출처와 면책

데이터 출처는 서울 열린데이터광장(data.seoul.go.kr)입니다. 공공 API 특성상 실제 현장 상황과 표시 정보가 일시적으로 다를 수 있으며, 본 서비스는 정보 제공만을 목적으로 합니다.
