1. 서비스 성격

따릉이 레이더는 서울 열린데이터광장에서 제공하는 공공 API를 활용해 대여소 정보, 대여 가능 수량, 대체 대여소 정보를 보여주는 비공식 정보 제공 서비스입니다. 서울시, 서울시설공단, 서울자전거 따릉이의 공식 서비스가 아닙니다.

2. 수집하는 정보

  • 현재 위치 정보: 주변 대여소를 계산하기 위해 브라우저 권한 허용 시에만 사용합니다.
  • 저장 장소 정보: 사용자가 입력한 장소명, 주소, 좌표, 알림 시간을 브라우저 저장소에 저장합니다.
  • 푸시 알림 정보: Android 앱에서 알림을 허용하면 기기 식별자, FCM 토큰, 저장 장소와 알림 시간이 서버에 저장될 수 있습니다.
  • 서비스 이용 로그: 오류 확인, 알림 발송 기록, 서비스 개선을 위해 필요한 최소 로그가 저장될 수 있습니다.

3. 이용 목적

  • 현재 위치와 저장 장소 주변의 대여소 상태 확인
  • 저장 장소 기준 레이더 카드와 대체 대여소 추천
  • 사용자가 설정한 알림 시간 안내
  • 서비스 오류 확인과 기능 개선

4. 보관 기간

저장 장소와 알림 설정은 사용자의 브라우저 저장소 또는 서버 DB에 보관될 수 있습니다. 사용자가 앱 데이터를 삭제하거나 알림을 해제하면 관련 설정이 비활성화될 수 있습니다. 알림 발송 기록은 오류 확인과 중복 발송 방지를 위해 필요한 기간 동안 보관합니다.

5. 제3자 제공 및 위탁

사용자의 저장 장소 정보를 제3자에게 판매하지 않습니다. 다만 지도 표시, 공공 API 조회, 서버 푸시, 배포 인프라 운영을 위해 카카오 지도, 서울 열린데이터광장, Firebase, Cloudflare Pages/Workers/D1 등의 외부 서비스가 사용될 수 있습니다.

6. 위치정보 안내

위치정보는 주변 대여소를 계산하기 위한 목적으로만 사용합니다. 현재 위치 권한을 거부하면 현재 위치 기반 기능이 제한될 수 있습니다. 저장 장소 알림을 사용하는 경우 사용자가 직접 저장한 장소 좌표가 알림 계산에 사용될 수 있습니다.

7. 문의

개인정보, 위치정보, 서비스 오류 관련 문의는 운영자 이메일로 접수합니다.

문의 이메일: leedeveloper79@gmail.com

8. 데이터 출처와 면책

데이터 출처는 서울 열린데이터광장(data.seoul.go.kr)입니다. 공공 API 특성상 실제 현장 상황과 표시 정보가 일시적으로 다를 수 있으며, 본 서비스는 정보 제공만을 목적으로 합니다.
