# Google Play 등록 문구 초안

## 앱 이름 (확정)

```text
따릉이 레이더
```

`따릉이`로 검색하는 사용자가 곧 이 앱의 사용자이므로, 영문명(`Bike Radar
Seoul`)으로는 검색 유입이 사실상 끊긴다. 웹·PWA·앱 모두 `따릉이 레이더`로
통일한다.

상표 관점에서는 `따릉이` 단독이 아닌 `따릉이 + 레이더` 조합이라 이 문서
초기판이 이미 허용 범위로 본 형태(`Bike Radar - 따릉이 알림`)와 같은 수준이다.
다만 공식 오인을 막는 장치는 반드시 함께 간다.

- 짧은 설명 첫머리에 `비공식`을 명시한다.
- 아이콘 배경은 남색을 쓴다. 따릉이 공식 브랜드가 초록 계열이므로 초록을
  주색으로 깔지 않는다.
- 아래 금지 표현을 설명문 어디에도 쓰지 않는다.

## 짧은 설명

```text
비공식 앱. 집·회사 주변 따릉이 재고와 대체 대여소를 정한 시간에 알려드려요.
```

## 긴 설명

따릉이 레이더는 서울 따릉이 이용자를 위한 비공식 정보 제공 앱입니다. 서울시가 만든 공식 앱이 아닙니다.

현재 위치와 저장한 장소 주변의 대여소를 확인하고, 대여 가능한 따릉이가 부족할 때 대체 대여소를 빠르게 찾을 수 있습니다. 집, 회사, 학교처럼 자주 쓰는 장소를 저장하면 지정한 시간에 주변 따릉이 상황을 푸시 알림으로 받을 수 있습니다.

주요 기능:

- 현재 위치 주변 따릉이 대여소 확인
- 저장 장소 2곳 기준 주변 재고 확인
- 대체 대여소 추천
- 지정 시간 Android 푸시 알림
- 서울 열린데이터 기반 대여소 이용 통계

데이터 출처:

- 서울 열린데이터광장

안내:

본 앱은 서울시, 서울시설공단, 서울자전거 따릉이의 공식 앱이 아닙니다. 공공 API 특성상 실제 현장 상황과 표시 정보가 일시적으로 다를 수 있습니다. 공식 이용권, 결제, 고장 신고, 민원은 공식 따릉이 서비스를 이용해 주세요.

## 스크린샷 문구 후보

- 저장 장소 주변 따릉이 상황을 한눈에
- 부족할 때 바로 보는 대체 대여소
- 집과 회사 기준으로 정해진 시간 알림
- 실시간 재고와 대여이력 통계

## 카테고리 후보

- 지도/내비게이션
- 라이프스타일

## 태그 후보

- 따릉이
- 서울 자전거
- 공공자전거
- 출퇴근
- 자전거
- 대여소

## 데이터 보안 입력 초안

- 위치정보: 주변 대여소 계산과 저장 장소 알림에 사용
- 앱 활동/기기 ID: 푸시 알림 토큰과 발송 기록 관리에 사용
- 개인정보 판매: 하지 않음
- 데이터 삭제 요청: 문의 이메일로 접수
