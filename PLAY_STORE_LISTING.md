## 짧은 설명

서울 따릉이 주변 재고와 저장 장소 알림을 한눈에 확인하세요.

## 긴 설명

Bike Radar Seoul은 서울 따릉이 이용자를 위한 비공식 정보 제공 앱입니다.

현재 위치와 저장한 장소 주변의 대여소를 확인하고, 대여 가능한 따릉이가 부족할 때 대체 대여소를 빠르게 찾을 수 있습니다. 집, 회사, 학교처럼 자주 쓰는 장소를 저장하면 지정한 시간에 주변 따릉이 상황을 푸시 알림으로 받을 수 있습니다.

주요 기능:

- 현재 위치 주변 따릉이 대여소 확인
- 저장 장소 2곳 기준 주변 재고 확인
- 대체 대여소 추천
- 지정 시간 Android 푸시 알림
- 서울 열린데이터 기반 대여소 이용 통계

데이터 출처:

- 서울 열린데이터광장

안내:

본 앱은 서울시, 서울시설공단, 서울자전거 따릉이의 공식 앱이 아닙니다. 공공 API 특성상 실제 현장 상황과 표시 정보가 일시적으로 다를 수 있습니다. 공식 이용권, 결제, 고장 신고, 민원은 공식 따릉이 서비스를 이용해 주세요.

## 스크린샷 문구 후보

- 저장 장소 주변 따릉이 상황을 한눈에
- 부족할 때 바로 보는 대체 대여소
- 집과 회사 기준으로 정해진 시간 알림
- 실시간 재고와 대여이력 통계

## 데이터 보안 입력 초안

- 위치정보: 주변 대여소 계산과 저장 장소 알림에 사용
- 앱 활동/기기 ID: 푸시 알림 토큰과 발송 기록 관리에 사용
- 개인정보 판매: 하지 않음
- 데이터 삭제 요청: 문의 이메일로 접수
