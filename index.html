홈 화면 추가 방법

iPhone Safari

  1. Safari에서 따릉이 레이더를 엽니다.
  2. 하단 공유 버튼을 누릅니다.
  3. 홈 화면에 추가를 선택합니다.
  4. 추가를 누르면 앱처럼 실행됩니다.

Android Chrome

  1. Chrome에서 따릉이 레이더를 엽니다.
  2. 오른쪽 위 메뉴를 누릅니다.
  3. 앱 설치 또는 홈 화면에 추가를 선택합니다.
  4. 설치를 누르면 앱처럼 실행됩니다.
