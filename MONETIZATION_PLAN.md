# 수익화 계획

## 현재 결론

초기 수익화는 AdMob 하단 배너 1개부터 시작한다. 전면 광고와 보상형 광고는 초기 UX에 맞지 않으므로 보류한다.

## 1단계: 준비

- 개발자 웹사이트 공개: 완료
- 개인정보처리방침 공개: 완료
- 서비스 이용약관 공개: 완료
- 위치정보 이용 안내 공개: 완료
- app-ads.txt 파일 공개: 완료
- AdMob 게시자 ID 발급 후 `app-ads.txt` 실제 라인 반영

## 2단계: AdMob

권장 광고 위치:

- 레이더 화면 최하단 배너
- 꿀팁/통계 화면 하단 배너

피해야 할 광고:

- 앱 시작 직후 전면 광고
- 지도 위를 가리는 광고
- 알림 권한 요청 직전/직후 광고
- 저장 버튼 주변 오클릭 유도 광고

## 3단계: 프리미엄 후보

- 광고 제거
- 저장 장소 2개에서 5개로 확장
- 장소별 알림 시간 여러 개
- 출근/퇴근 고급 알림
- 통계 고급 카드

## 4단계: 웹 수익화

SEO 콘텐츠가 충분히 쌓인 뒤 AdSense를 검토한다. 현재 앱 셸 중심 사이트는 AdSense 심사용 콘텐츠가 부족할 수 있으므로, 지역/역/출퇴근 가이드 페이지를 먼저 늘린다.
