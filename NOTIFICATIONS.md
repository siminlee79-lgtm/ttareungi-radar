# 알림 전략

따릉이 레이더는 알림이 핵심 기능입니다. MVP에서는 PWA Web Push를 기본 알림으로 쓰고, 서비스 검증 후 카카오 알림톡 또는 문자 알림을 보조 채널로 붙입니다.

## 1순위: PWA Web Push

- 사용자가 PWA에서 알림을 허용하면 브라우저 푸시를 보냅니다.
- Android Chrome은 비교적 자연스럽게 동작합니다.
- iPhone은 iOS 16.4 이상에서 홈 화면에 추가한 PWA에 Web Push를 보낼 수 있습니다.
- 무료로 시작 가능하지만, 실제 예약 알림은 서버와 푸시 구독 저장 DB가 필요합니다.

## 2순위: 카카오 알림톡

- 출근 전 위험 알림처럼 서비스가 사용자에게 자동으로 보내는 메시지는 일반 카카오톡 메시지 API보다 알림톡 성격에 가깝습니다.
- 카카오 알림톡은 비즈니스 채널, 템플릿 승인, 발송 비용, 대행사 연동이 필요합니다.
- MVP 검증 뒤 유료 알림 또는 프리미엄 기능으로 붙이는 편이 좋습니다.

## 3순위: 문자 또는 이메일

- 문자는 도달률이 높지만 발송 비용이 있습니다.
- 이메일은 저렴하지만 출근 전 즉시성은 약합니다.

## 추천 순서

1. PWA 내부 알림 허용 UI 만들기
2. Web Push 서버 붙이기
3. 사용자가 실제로 알림을 원하는 시간대 수집
4. 카카오 알림톡 도입 여부 결정
