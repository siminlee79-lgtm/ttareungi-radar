# 따릉이 레이더

내가 매일 쓰는 대여소가 오늘 몇 시에 위험한지 미리 알려주는 따릉이 예보 PWA입니다.

## 지금 들어간 것

- 모바일 우선 PWA 화면
- 설치용 `manifest.webmanifest`
- 오프라인 캐시용 `service-worker.js`
- 알림 허용 UI와 테스트 알림
- 현재 날짜와 시간 기준 위험도 계산
- 현재 위치 기준 지도 흐름
- 마이메뉴 장소 저장: 집, 사무실, 학교, 학원, 기타
- 저장한 도착지 기준 목적지 주변 상황 예보
- 임시 대여소 데이터 기반 위험/주의/안전 표시

## 로컬 실행

```powershell
python -m http.server 5173
```

브라우저에서 `http://localhost:5173`을 엽니다.
